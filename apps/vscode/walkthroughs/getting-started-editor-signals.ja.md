# エディターの表示を読む

VS Code は language feature をエディター上に直接表示します。

- 赤、黄色、薄い下線は diagnostics です。下部 panel の Problems を開くと一覧で確認できます。
- 入力中に候補が出ます。手動で候補を出す場合は `Ctrl+Space` を使います。macOS では環境によって `Cmd+Space` が OS の入力切り替えに使われるため、Command Palette から Trigger Suggest を実行しても構いません。
- keyword、built-in object、variable、function、sub、class、include path に mouse cursor を重ねると hover 情報が出ます。
- project 内を移動したいときは、右クリック menu から Go to Definition や Find All References を使います。
- Classic ASP と embedded HTML、CSS、JavaScript、VBScript を整えるときは Format Document を使います。

最初から command を全部覚える必要はありません。多くの機能は、右クリック menu、Command Palette、diagnostics や refactor の近くに出る light bulb から使えます。
