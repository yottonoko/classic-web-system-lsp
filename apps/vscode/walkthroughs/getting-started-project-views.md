# Use Project Views

Classic ASP projects often depend on include files and shared VBScript routines. Classic ASP LSP includes a few views that help you inspect a project without reading every file manually.

- Project glob files: preview which files are included in workspace analysis, then adjust include and exclude globs.
- Current file graph: show how the current file is connected to includes, declarations, references, assignments, and calls.
- Folder or workspace graph: inspect a wider area when a project has many includes.
- Current file flowchart: read control flow for the active ASP file.
- Excel export: export current file analysis when you need a shareable workbook.

Open these from the Command Palette by searching for `Classic ASP`. Some commands also appear in the editor title or Explorer context menu for `.asp`, `.asa`, and `.inc` files.
