# Open a Classic ASP File

Classic ASP LSP starts when VS Code opens a supported file:

- `.asp`, `.asa`, `.inc` use the `classic-asp` language mode.
- `.vbs` uses the `vbscript` language mode.

If you are new to VS Code, open the Explorer from the Activity Bar on the left, select a file, and check the language name in the lower-right Status Bar. When the language mode is correct, this extension can provide diagnostics, completions, hover text, navigation, formatting, CodeLens, and inlay hints.

For `.inc` files, the extension treats the file as a fragment. It does not assume the include file is a complete HTML document.
