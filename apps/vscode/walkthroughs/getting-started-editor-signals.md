# Read Editor Signals

VS Code shows language features directly inside the editor.

- Red, yellow, or faint underlines are diagnostics. Open the Problems panel to see them in a list.
- Suggestions appear while typing. Press `Ctrl+Space` (`Cmd+Space` on macOS if available) to request suggestions manually.
- Hover over a keyword, built-in object, variable, function, sub, class, or include path to see more information.
- Use Go to Definition or Find All References from the right-click menu when you want to move through a project.
- Use Format Document when you want asp-lsp to format Classic ASP with embedded HTML, CSS, JavaScript, and VBScript.

You do not need to learn every command first. Most features are available from the editor context menu, the Command Palette, or the light bulb that appears near diagnostics and refactor suggestions.
